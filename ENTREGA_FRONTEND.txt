🚀 COMO COMEÇAR
═════════════════════════════════════════════════════════════════════════════

1️⃣  Iniciar Backend
    cd backend
    npm install
    npm start

2️⃣  Iniciar Frontend (Terminal 2)
    cd frontend
    python -m http.server 8000

3️⃣  Abrir no Navegador
    http://localhost:8000

4️⃣  Fazer Login
    Usuário: Admin
    Tipo: admin
    Clique em "Entrar"

5️⃣  Explorar!
    • Dashboard: Veja estatísticas
    • Ordens: Filtros e listagem
    • Usuários: Gerenciar usuários
    • Admin: Importar dados e distribuir

  ├─ id (ID único)
